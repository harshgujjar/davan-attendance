Core Features
📊
Attendance Dashboard
Real-time student attendance from college portal. Class-wise breakdown, subject-wise percentage bars, absentee tracking, defaulter lists below 75%.
🏖️
Leave Manager
Faculty leave with 4-tier substitute chain. Auto-finds free faculty, assigns substitutes per period, WhatsApp message generator, compensation tracker.
📝
Internal Timetable
Create/edit internal exam timetables with per-date time slots. Subject dropdown auto-filled from allocation data. Duty allocation per slot with serial numbers. PDF + Excel download.
📅
Timetable & Allocation
View, edit, and manage class timetables and faculty allocations. Subject Index with code mapping. Lab batch handling. Unmatched code detection.
📣
Live Banner System
3 banner channels: Admin broadcast (Banner 1), Holiday ticker (Banner 2), Internal duty reminder (Banner 3). Real-time push to all open devices via Firebase.
🔔
Faculty Alerts & Push
Day-before push notifications to faculty. Configurable per faculty. Test notification broadcaster. Subscription management per device.
📰
Academic Calendar
Holiday and event management with types (holiday, exam, event, meeting). Integrates with Leave Manager to block exam dates. Visual calendar view.
📋
Results Module
Separate results.html for student result cards. Subject-wise marks, pass/fail status, percentage breakdown. Linked from main app sidebar.
👥
User Management
Create/edit/delete user accounts. Role-based access control. Password reset flow. Login activity tracking. Secondary Firebase app for safe account creation.
🤖
Smart Assist (New)
Rule-based intelligence. Substitute suggester with 6 scoring rules. Attendance + leave + workload alerts. Plain English search for students, faculty, leave records.

R5 4+ subs this week → −25 pts
R6 2–3 subs this week → −10 pts
